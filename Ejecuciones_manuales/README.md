# Ejecuciones manuales

Carpeta para `.md` con SQL, deploys u otras operaciones que se ejecutan a mano
(nunca desde Claude/CI). El contenido está gitignored — solo este README se
versiona para conservar el directorio.

Formato (patrón de sozu-admin): título con contexto del cambio, secciones
fechadas (`## Paso X — descripción — YYYY-MM-DD`) y bloques de código con los
comandos exactos listos para copiar.
